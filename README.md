# www.prosegrammers.com.

This is the GitHub repository for
[Prosegrammers](http://www.prosegrammers.com/), a website for a course about
document engineering. Prosegrammers learn to use Python for creating,
analyzing, and improving documents, blending programming with technical
writing. The course covers tools like Jupyter Notebooks and Quarto, and
teaches skills in documentation, version control, and AI-assisted coding
with [GitHub Copilot](https://github.com/features/copilot) and
[OpenCode](https://opencode.ai/). All resources and projects are publicly
available.

The website serves as a comprehensive hub for students enrolled in the
Document Engineering course, offering structured access to course materials,
assignments, and interactive learning experiences. It emphasizes hands-on
practice with real-world tools and fosters collaboration among
prosegrammers—individuals who master both coding and documentation.

## Site Contents and Contributions

- **Syllabus**: Detailed course outline covering objectives, policies, and
  grading.
- **Weekly Schedule**: Organized assignments and readings for each week of
  the semester.
- **Lecture Slides**: Interactive presentations built with Quarto and
  reveal.js, featuring code examples and exercises.
- **Code Examples and Projects**: Python scripts, Jupyter notebooks, and
  document processing tools to illustrate key concepts.
- **Documentation Resources**: Guides on best practices for writing clear,
  professional documentation and using version control.
- **Open Educational Resources**: Free access to materials that contribute
  to the broader field of document engineering, promoting accessibility and
  innovation in technical education.
